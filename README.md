# Riva Refuge Website

Source for [www.rivarefuge.org](https://www.rivarefuge.org) — a small, secular non-profit helping children, youth, young women, disabled, and elderly people in Africa and beyond.

Built with [Astro](https://astro.build) and deployed to GitHub Pages via GitHub Actions on every push to `master`.

## Development

```bash
npm install
npm run dev      # http://localhost:4321/
npm run build    # builds to dist/
```

## Stack

- [Astro 5](https://astro.build) — static site generator
- [Bootstrap 5.3.2](https://getbootstrap.com) — CSS framework (CDN)
- [Font Awesome 6.4.0](https://fontawesome.com) — icons (CDN)
- GitHub Pages — hosting
- GitHub Actions — CI/CD
